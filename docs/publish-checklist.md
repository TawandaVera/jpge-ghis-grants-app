# Publish checklist

Before publishing from Base44:

- [ ] PR merged into `main`.
- [ ] `npm run lint` passed.
- [ ] `npm run test -- --run` passed.
- [ ] `npm run build` passed.
- [ ] Assessment date wiring confirmed.
- [ ] Upload guard wiring confirmed.
- [ ] Auth mode confirmed.
- [ ] Admin route access confirmed.
- [ ] LLM prompt data boundary confirmed.
- [ ] Source evidence verification confirmed.
- [ ] Pack export gate confirmed.
